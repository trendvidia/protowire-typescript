# Code of Conduct

We expect participation in this project — issues, pull requests,
discussions, comments — to be respectful, technical, and on-topic.

Personal attacks, harassment, dismissive language, and bad-faith
arguments are not welcome and will be moderated.

To report a concern, email **conduct@trendvidia.com**. Reports are read
by the project's stewards and acknowledged within 3 business days.

This policy mirrors the upstream
[`CODE_OF_CONDUCT.md`](https://github.com/trendvidia/protowire/blob/main/CODE_OF_CONDUCT.md);
if anything here disagrees with the upstream version, the upstream wins.
